Exercise tracker API

Create User

To create a new user, provide a username and submit the form below. The server will respond with the created user's details.

POST /api/users

Get Users

Click the button below to retrieve a list of all users from the server. The result will be displayed below the button.

GET /api/users

Create Exercise

To create a new exercise for a user, provide the user ID, exercise description, duration, and date. Submit the form to create the exercise.

POST /api/users/:userId/exercises

Get Logs

To get logs for a specific user, provide the user ID and optional filters (from date, to date, and limit). Submit the form to fetch the logs.

GET /api/users/:userId/logs
